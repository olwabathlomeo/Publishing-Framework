# Writing Guide

Use this project as a clean workspace for writing chapters.

## Recommended workflow

1. Open a chapter file in `chapters/`.
2. Copy the structure from `templates/chapter-template.md`.
3. Write your content in Markdown.
4. Run `./scripts/build.sh` to generate the book.
5. Review the output in `output/`.

## Tips

- Keep paragraphs short.
- Use headings to break content into readable sections.
- Add examples and exercises where helpful.
- Rebuild often while editing.
