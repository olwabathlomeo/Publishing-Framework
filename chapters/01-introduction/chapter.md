# Introduction

This book is a starter example for a publishing workflow built around plain Markdown files and Pandoc. The goal is to make it easy to write, organize, and publish content in multiple formats without requiring a complex publishing system.

## Why this framework matters

A simple framework helps authors focus on the content rather than the tooling. With a predictable file structure, a small set of templates, and a single build command, it becomes straightforward to generate:

- HTML for the web
- PDF for print or offline sharing
- future outputs such as slides or EPUB

## Structure of the project

The repository is organized into a few key areas:

1. Frontmatter for title pages and metadata
2. Chapters for the main content
3. Templates for reusable formats
4. Output for generated books

This structure provides a practical foundation for expanding the project into a more complete book or course pack.
