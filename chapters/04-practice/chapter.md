# Practice and Review

A good publishing workflow should support regular review. This chapter introduces a simple practice cycle: draft, review, revise, and publish.

## Suggested workflow

1. Write the chapter content in Markdown.
2. Build the HTML and PDF outputs.
3. Review the outputs for clarity and formatting.
4. Revise the source files and rebuild.

This loop makes it easy to improve the book incrementally without needing a large setup.
